XSL adalah style sheet untuk XML, tapi tidak sekedar itu karena terdiri dari 4 Bagian:
	1. XSLT		: bahasa untuk transformasi document XML menjadi XHTML atau document XML lain
					XSLT(XSL Transformasi)
	2. XPath	: bahasa untuk navigasi di document XML
	3. XSL-FO	: bahasa untuk formatting document XML(berhenti di 2013)
	4. XQuery	: bahasa untuk untuk querying document XML
