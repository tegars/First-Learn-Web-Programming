XML Attribute Values Must be Quoted contoh salah: <note date=12/11/2007> <to>Tove</to> <from>Jani</from> </note> contoh benar: <note date="12/11/2007"> <to>Tove</to> <from>Jani</from> </note> 05. Entity References jangan memasukkan simbol ini di XML, jadi harus diganti &lt; < less than &gt; > greater than &amp; & ampersand &apos; ' apostrophe &quot; " quotation mark 06. Comments in XML <!-- This is a comment --> 07. XML Stores New Line as LF Windows applications store a new line as: carriage return and line feed (CR+LF). Unix and Mac OSX uses LF. Old Mac systems uses CR. XML stores a new line as LF.
